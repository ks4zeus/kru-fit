# Assets

Drop the Kru Fit logo here. Suggested files:

- `logo.svg` — primary logo (preferred; scales cleanly)
- `logo.png` — fallback raster, ~512×512
- `favicon.png` — 32×32 or 48×48 (browser tab icon)
- `apple-touch-icon.png` — 180×180 (iOS home-screen icon)

Once a file is here it's served by Pages at `/assets/<filename>`
(e.g. `/assets/logo.svg`). Tell me the filename you used and I'll wire it
into the header (replacing the ⚡ emoji) and add the favicon/touch-icon links.
